# 📸 Meme Gallery - Day 11 (CSS Flexbox Challenge)

Welcome to **Day 11** of my web development journey under the **"WEB UNLEASHED: From Hacker to Master"** project.

## 🔥 What I Built

This day's challenge was to create a **responsive Flexbox-based image gallery** layout that displays memes in a 2-row format. It adapts based on screen size and includes spacing between images.

## 🧠 Concepts Used

- CSS Flexbox (`display: flex`, `flex-wrap`, `justify-content`, `gap`)
- Image scaling using `object-fit: cover`
- Responsive row structure (2-per-row target)
- Custom CSS styling & layout logic
- Debugging & Visual consistency

## 🧪 Output Preview

Each meme appears inside a styled `div.card` and is displayed in a row with consistent sizing and spacing.

## 📂 Files Included

- `index.html` – HTML structure for the gallery
- `style.css` – Flexbox layout styling
- `images/` – Folder containing all meme images
