# Day 14: Responsive Flexbox Feature Layout 🚀

This project is part of my "WEB UNLEASHED: From Hacker to Master" journey.

## 📚 What I Learned:
- How to build structured UI layouts using Flexbox
- Aligning content side-by-side using `.feature-row`
- Creating two types of content in each row (e.g., features + plan)
- Making the layout responsive using `@media` queries

## 🧠 Real-World Use:
This type of layout is commonly used in:
- Pricing plan comparisons
- Feature highlights on SaaS product pages
- Landing pages with visual + text content blocks

## 💻 Tech Used:
- HTML5
- CSS3 Flexbox
- Responsive Media Queries

## ✅ Output Preview:
- Clean two-column layout on desktop
- Stacked content on mobile
- Stylish, modern section for any landing page or portfolio
