# 🛍️ Product Showcase – Day 13 (Responsive Flexbox Cards)

Welcome to **Day 13** of my web development journey under the _"WEB UNLEASHED: From Hacker to Master"_ project.

---

## 🎯 What I Built

This challenge was focused on creating a **responsive product card layout** using **CSS Flexbox**.  
The goal was to display multiple products (like T-shirts, hoodies, caps) in a clean, mobile-friendly grid.

Each card includes:

- 📸 Product image  
- 📝 Product name  
- 💰 Price  
- 🛒 Add to Cart button

---

## 🧠 Concepts Used

- CSS **Flexbox** layout  
- Responsive design using `flex-wrap`, `gap`, and `justify-content`  
- Custom card UI with padding, rounded corners, and shadow  
- Hover transition effects  
- Mobile responsiveness with `@media` queries  
- Visual hierarchy using typography & spacing

---

## 📱 Real-World Use

This layout can be used in **e-commerce websites**, clothing stores, digital product pages, etc.  
It’s the base structure for building a fully functional **product grid UI**.
